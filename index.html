Water Bringers

charity:water's Mission:
charity:water is a non-profit organization bringing clean and safe drinking water to people in developing countries.
So in an effort to spread awareness for the good work they do, this game works to give you a window into some of the infrastructural challenges faced by charity:water.
Learn more at charitywater.org.

Game Goal:
Sustain all enabled regions by providing the water solutions they need before time runs out.

How to Play:

  • Select an action from the sidebar (such as Build Well or Install Rainwater Catchments).
  • Click a region (hex) to apply the selected action. Only needed actions can be applied to each region.
  • Each region has different needs. Complete all needs to sustain a region.
  • Some regions are randomly disabled each game for replayability.
  • You gain $100 in funding every 10 days. Use funding to perform actions.
  • Complete all enabled regions before time runs out!
